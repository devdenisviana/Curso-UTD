Conhecendo as principais tags

O HTTP é o protocolo de rede que permite a transferência de arquivos, ao passo que a World Wide Web (WWW) é o sistema que cataloga e permite compartilhar documentos por meio do HTTP. Basicamente, o HTML é a unidade essencial de todas as construções da web. Toda página na internet começou meramente como um documento em HTML, com as tags básicas estruturando e formatando esse documento.
